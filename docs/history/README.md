# docs/history/ — superseded design & spec documents

These predate the current product surface and are kept for provenance. They are **not** the
authoritative description of what the workbench does today — for that, see the root `README.md`,
`docs/USER_GUIDE.md`, and `docs/MANIFOLD.md`.

- **`goal.md`** — the original feature contract for the **Steering Bench + Recipe Cards +
  Autopilot** layer. Those features shipped and are still part of the product, but this document
  predates the Lab Bench web-UI redesign and the manifold work, so it describes the older
  Gradio-tabs framing. Useful as the historical spec for the benchmark/recipe/autopilot semantics.
- **`gui_redesign_brainstorm.html`** — the Phase-0 brainstorm that proposed the three design
  directions (Lab Bench / Feature Atlas / Steering Studio) and the recommendation to fuse them.
- **`lab_bench_prototype.html`** — the Phase-0 static mockup of the fused Lab Bench on mock data.
  It was the design source for the live `web/` SPA, which has since superseded it.

For the dead-end *research code* (the negative-result geometry investigation), see `archive/`.
